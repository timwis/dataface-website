Portability matters

Databases built with Dataface are regular SQL databases, which provides several benefits.

No confusing, proprietary data structure A column in the application is a column in the database.

Easy to integrate with other applications Connect the other application directly to the database.

You control the database Use our hosted offering or point Dataface to your cloud or data center.

Ready for you to outgrow it Stop using Dataface at any point and just build an application on top of its database as you normally would.

Use with existing databases Since Dataface aims to build as vanilla a database as possible, it works with existing databases too.

Leverage the full power of databases Queries, triggers, row-level security, replication, indexes, and more.
